PTRDUMP.pyDumps PTR (Reverse IP lookup) records for a range of IP addresses. Can be used in internal domain networks as well.Switches :(-r) Specify a range of IP addresses. Octets are broken down into reverse order and converted into an in-addr.arpa. domain names.(-s) Specify specific DNS servers to question, ignoring the default list of DNS servers.(-f) Specify the path to a file (such as a firewall log file) that contains IP addresses. Results are ordered by number of times the IP is seen in descending order.(-arin) Query ARIN's API to identify the OrgName and Net Name associated with the given IP.Examples :Question 1.1.1.1 and 8.8.8.8 for the reverse ordered IP's between 18.4.86.1-50python ptrdump.py -r 18.4.86.1-50 -s 1.1.1.1,8.8.8.8Domain not available for 18.4.86.1========================================Domain not
